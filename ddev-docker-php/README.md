# DDEV Docker + PHP

### What is DDEV? 
Docker-based development environments.

### Which environments are supported?
* PHP
* Python
* Node.js

### Quick Start

* install docker
* install DDEV
* ready!

### How to create a project?

* go to a folder
```
cd ~/dev/my-project
```
* initialize
```
ddev config
```
* start
```
ddev start
```
* see the projectâ€™s parts
```
ddev describe
```
* run
```
ddev launch
```
* done!
